# Runtime HRMS Logo

## Add Your Logo Here

Please add the Runtime HRMS logo file to this directory:

**File name:** `runtime-logo.png`

**Location:** `app/static/images/runtime-logo.png`

## Logo Specifications

- **Format:** PNG (with transparent background preferred)
- **Recommended size:** 400px width x 120px height
- **Content:** Blue "RUNTIME" text + Green "HRMS" text + Blue curved icon
- **Background:** Transparent or white

## How to Add the Logo

### Option 1: Download from Runtime HRMS Website
1. Go to the Runtime HRMS website
2. Right-click on the logo
3. Save image as `runtime-logo.png`
4. Place it in this directory

### Option 2: Use Your Own Logo File
1. Prepare your logo file (PNG format)
2. Name it `runtime-logo.png`
3. Place it in this directory: `app/static/images/runtime-logo.png`

### Option 3: Create from the Image You Provided
